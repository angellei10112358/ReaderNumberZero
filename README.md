# 永夜书馆·零号借阅者

> 十四页的命运 · 十三座书馆 · 七个结局

一款中式哥特风格的文字冒险游戏（hypertext fiction），纯前端，打开即玩。

---

## 快速开始

直接打开 `index.html` 即可开始游戏。无需安装、无需服务器、无需网络（Google Fonts 字体加载需要网络，不影响核心功能）。

---

## 故事背景

在平行的"墨海纪元"，人类拥抱"文字即真实"的终极法则。永夜书馆——十三座漂浮在墨海之上的巨大图书馆——记录着一切被书写和被阅读的故事。

你是**零号借阅者**。三年前，你在第零馆借阅了《你的死亡》，读完后把它烧了。从那以后，你成了"被命运通缉的人"，现实不断出现"错行"（叙事层面的故障）。

今夜，第零馆的封印裂开了。一本从未被记录的书从裂缝中飘落，封面上写着：

> **「借阅者，你终于来了。这一次，轮到你被阅读了。」**

你的剩余页数为 **14 页**——这是你还能"存在"的寿命。每使用一次能力，页码就会减少。在页码归零之前，你必须穿过十三座主馆，找到正在写你的作者，做出最终的选择。

---

## 游戏机制

### 页数系统
- **剩余页数** = 你的生命值 + 行动资源
- 每次使用「删改」能力消耗 2–3 页
- 每次使用「窥视」能力消耗 1 页
- 页数归零则游戏结束（但「此刻之锚」可救一次）

### 能力系统（一次性）
- **「删改」**：修改一段现实，改变故事走向
- **「窥视」**：看到隐藏信息和深层结构

### 能力升级（通过剧情获得）
- **烈火书页**：用火焰销毁现实（洛言之书）
- **存在抹除**：从因果层面删除目标（消耗最大页数）
- **叙事治愈**：讲述一个完整的故事来恢复 1 页
- **遗忘诗句**：看到规则的缝隙（疯诗人）
- **禁书真知 / 真实之眼**：看到事物的叙事本质（第八馆）
- **此刻之锚**：页数归零时自动恢复 1 页（苏眠，一次性）

### 章节系统
随着你探索不同的书馆，章节会自动推进（第一馆→第十四馆），反映剧情的深度进展。

## 结局（共 7 个）

| 结局 | 最短路径 | 难度 | 说明 |
|------|----------|------|------|
| 全书完 | 4 场景 | ⭐ 极简 | 终结一切故事 |
| 故事解放者 | 4 场景 | ⭐ 极简 | 释放所有被囚禁的故事 |
| 新作者 | 5 场景 | ⭐ 简单 | 取代作者，重写世界 |
| 撕书人 | 5 场景 | ⭐ 简单 | 撕毁命运，获得自由 |
| 规则重写者 | 5 场景 | ⭐⭐ 中等 | 改写叙事底层规则 |
| 二重身 | 5 场景 | ⭐⭐ 中等 | 过去、现在、未来的自己相遇 |
| 守密人 | 6 场景 | ⭐⭐⭐ 唯一 | 焚烧真相成为沉默守护者（需烈火书页路线） |

所有结局均可零页数到达。页数系统的设计优先叙事自由度而非资源压力。

## Glitch 事件（错行）

游戏中约 8% 的场景转换会随机触发 glitch 事件（前提是页数 > 3）。这些叙事故障会短暂打断当前场景，展示书页的自我意识——它们是被书写世界中的"现实 bug"。

## 技术说明

- 纯 HTML + CSS + JavaScript，无外部依赖
- 所有场景数据、游戏状态、渲染逻辑在单个文件中
- 约 3100 行，145+ 个场景，全文约 10 万字叙事内容
- 游戏引擎基于场景对象 + 选择渲染循环（类似 ChoiceScript 风格）

### 文件结构

```
reader_number_zero/
├── index.html    # 游戏本体（打开即玩）
├── game.html     # game.html 与 index.html 内容一致（备用）
├── bg.txt        # 世界观背景原文
└── README.md     # 本文件
```

## 开发说明

### 场景结构

每个场景是一个 `scenes.sceneId = {}` 对象：
- `render()` → 返回 HTML 字符串（叙事实体）
- `postRender()` → 副作用（设置 flags、扣除页数等）
- `choices[]` → 选项列表（text / next / hint / ability）

### 状态管理

`state` 对象持有全部游戏状态：
- `pages` / `maxPages`：页数
- `flags{}`：布尔标记（获得的能力、见到的角色等）
- `deleteUsed` / `peekUsed`：能力使用状态（一次性）
- `chapter`：当前章节（1–14）
- `history[]`：最近 10 个场景的浏览记录
